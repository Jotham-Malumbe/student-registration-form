# Student Registration Form

## Overview
This is a **simple student registration form** built with **HTML and CSS**.  
The form collects basic student information such as:

- Full Name
- Email
- Password
- Age
- Gender

It uses **HTML built-in validation** to ensure users enter correct and complete data.  
The project is beginner-friendly and a good practice for learning **forms, styling, and validation**.

## Features
- **Required fields**: All fields must be filled before submitting  
- **Email validation**: Ensures a valid email format  
- **Password validation**: Minimum 8 characters  
- **Age validation**: Only numbers accepted  
- **Dropdown validation**: Gender must be selected  
- **Input feedback**: Red borders for invalid inputs, green for valid inputs  
- **Responsive form layout**: Centered form with clean styling
